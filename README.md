# LaptopRentalSystem
📘 Laptop Rental System (C Project)
Overview
This project is a console-based Laptop Rental Management System built using C.
It is designed to manage customer rental data using:
•	Structures (struct)
•	File handling
•	CRUD operations
•	Real-world business logic (payments, dues, reminders)
________________________________________
Learning Goals
•	Master struct in C
•	Learn file handling (fopen, fread, fwrite)
•	Implement CRUD operations
•	Handle real-world data modeling
•	Practice input validation and control flow
